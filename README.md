# gkd-cli

> My template generator.

## Installation

```bash
$ npm install --global gkd-cli
```

## Usage

```bash
$ gkd app-name [options]
$ gkd --help
```

## Example

```bash
$ gkd test -t react-ts-webpack-template

  ✔ Copy files
  ✔ Success

Installation:

> cd test
> npm install

Scripts:

> npm run dev
> npm run build
> npm run prettier
```
